# START Menu

The START menu allows you to initiate JoinMarket-NG services as either a maker (liquidity provider) or taker (coinjoin initiator).

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                        JOININBOX - START MENU                                │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│        [MAKER]       Run as a maker to provide liquidity                     │
│                                                                              │
│        [TAKER]       Run as a taker to initiate coinjoin transactions        │
│                                                                              │
│        [ORDERBOOK]   View current orderbook                                 │
│                                                                              │
│        [BACK]        Return to main menu                                    │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

## Maker Mode

Run JoinMarket-NG in maker mode to provide liquidity for coinjoin transactions and earn fees.

## Taker Mode

Run JoinMarket-NG in taker mode to initiate coinjoin transactions for improved privacy.

## Orderbook

View the current live orderbook of available makers and their offers.
