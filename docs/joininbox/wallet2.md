# WALLET Menu

The WALLET menu provides functions for managing your JoinMarket-NG wallets and funds.

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                        JOININBOX - WALLET MENU                               │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│        [CREATE]      Create a new wallet                                    │
│                                                                              │
│        [LOAD]        Load an existing wallet                                │
│                                                                              │
│        [DISPLAY]     Show wallet information and balances                   │
│                                                                              │
│        [SEND]        Send funds (with or without coinjoin)                  │
│                                                                              │
│        [RECEIVE]     Generate receive addresses                             │
│                                                                              │
│        [LOCK]        Lock wallet funds                                      │
│                                                                              │
│        [UNLOCK]      Unlock wallet funds                                    │
│                                                                              │
│        [BACK]        Return to main menu                                   │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

## Create Wallet

Generate a new JoinMarket-NG wallet with secure encryption.

## Load Wallet

Load an existing wallet file to access your funds.

## Display Wallet

Show detailed wallet information including balances in each mixdepth.

## Send Funds

Initiate transactions with optional coinjoin for enhanced privacy.

## Receive Funds

Generate new receive addresses for funding your wallet.

## Lock/Unlock Funds

Securely lock wallet funds when not in use or unlock them for transactions.
